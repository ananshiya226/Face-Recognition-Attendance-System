# Face-Recognition-Attendance-System
The Face Recognition Attendance System automatically detects and identifies a person’s face using a webcam. It compares the detected face with stored images in the system. If the face matches, the system records the person’s name, date, and time as attendance. The project is built using Python, OpenCV, and face_recognition.

# Face Recognition Attendance System

This project automatically detects and recognizes a person's face using a webcam and marks attendance.

## Technologies Used
- Python
- OpenCV
- face_recognition
- NumPy
- Pandas

## Features
- Detect face using webcam
- Recognize registered faces
- Automatically mark attendance
- Store attendance with date and time

## How to Run
1. Install required libraries
pip install -r requirements.txt

2. Run the program
python main.py
